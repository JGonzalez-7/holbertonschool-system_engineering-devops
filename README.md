# System Engineering & DevOps

This repository contains system engineering and DevOps projects completed as
part of the Holberton School curriculum.

## Projects

### Web Infrastructure Design

The [`web_infrastructure_design`](web_infrastructure_design/) directory
contains Mermaid diagrams that illustrate how a web infrastructure can evolve
from a single-server setup into a distributed, secured, monitored, and scalable
architecture.

The project covers:

- DNS and domain name resolution
- Web and application servers
- MySQL databases and primary-replica replication
- Load balancing with HAProxy
- Firewalls, HTTPS, and SSL termination
- Infrastructure monitoring
- Redundancy and horizontal scaling

See the [project README](web_infrastructure_design/README.md) for details about
each design.
